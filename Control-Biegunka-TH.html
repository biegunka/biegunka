biegunkaOptions :: Name -> Q [Dec]Source

Make command line parser for biegunka library and script environments options

The following options become available:

  • --safe-run (default)
  • --run
  • --dry-run
  • --check
  • --full
  • Also one option for each environment (the --lowercased environment name)

Supports only simple sum types. (Simple here means non-empty, non-parametrized.)

The usage is trivial: 

 data Environments = X220 | T510

 biegunkaOptions ''Environments

 main :: IO ()
 main = do
   (environment, runBiegunka) <- options
   case environment of
     X220 -> runBiegunka ...
     T510 -> runBiegunka ...

transformString :: String -> StringSource

Transform data constructor name into command line option 

>>> transformString ""
""

>>> transformString "foo"
"foo"

>>> transformString "Foo"
"foo"

>>> transformString "FooBarBaz"
"foo-bar-baz"
