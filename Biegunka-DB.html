save :: Controls -> Biegunka -> IO ()Source

Save profiles data to files.

Each profile is mapped to a separate file in appData directory. Mapping rules are simple: profile name is a relative path in appData.

For example, profile dotfiles is located in ~/.biegunka/dotfiles by default and profile my/dotfiles is located in ~/.biegunka.my/dotfiles by default.
